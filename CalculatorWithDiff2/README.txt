Задача: реализовать калькулятор для вычисления арифметических выражений, упрощения выражений содержащих переменные и вычисления производных полиномиальных функций.

Программа должна обеспечивать следующую функциональность:
1) Чтение входных данных из файла.
2) Способность вычислять арифметические выражения. Выражение может состоять из целочисленных констант, переменных, арифметических выражений и арифметических операций (+,-,*,/), круглых скобок.
Имена переменных могут содержать только строчные буквы латинского алфавита без разделителей.
3) Возможно задавать значение переменных следующии образом: Repl(x, 10), Repl(ab, 20). В круглых скобках задается переменная, и далее ее значение.
Если значения всех переменных заданы, то калькулятор вычисляет значение такого выражения. Если не все переменные заданы, то происходит упрощение выражения с учетом заданных значений переменных.
4) Выражения, в которые входят переменные, определют полиномы. Допускаются выражения с отношением полиномов. При возможности выражения упрощаются - вычисляются коэффициенты перед переменными.
5) Дифференцирование выражений по заданной
переменной. Операция дифференцирования во входных данных имеет вид: D(x), с указанием в круглых скобках переменной, по которой производится дифференцирование.
Для полинома, полученного путем дифференцирования, также возможно задание переменных и упрощение или вычисление значения полиномиальной функции на этих переменных.
6) Вывод в выходной файл результата вычислений.

Входные данные.
Во входном файле задаются в отдельных строках выражения и операции, которые необходимо выполнить.
Операции над выражением должны задаваться в следующей строке; набору операций должны предшествовать символы "Operations:", например:
Expression: (2*4+x)*(8+y*y+z)+5
Операции над выражением должны задаваться в следующей строке после символов "Operations:", например:
Operations: D(x), D(ba), Repl(x, 10), Repl(ba, 20)
Операции над выражением выполняются в том порядке, в котором они заданы в файле, т.е. слева направо.

Выходные данные.
Выходной файл должен содержать результаты вычисления выражений.
Если для некоторого выражения данные некорректны, то в выходном файле в соответствующей для него строке выводится "INCORRECT".

Пример входных данных:
Expression: 2*x+x*x-3-(4*x-5)*(4*x-5)*(4*x-5)
Operations: D(x), Repl(x, 1)
Пример выходных данных:
-8
